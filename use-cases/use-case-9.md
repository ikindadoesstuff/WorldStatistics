# USE CASE: 8 View Population of Chinese, English, Hindi, Spanish, and Arabic Speakers Worldwide.

## CHARACTERISTIC INFORMATION

### Goal in Context

As a data analyst, I want to view the number and percentage of people who speak Chinese, English, Hindi, Spanish and 
Arabic worldwide, sorted from the highest to lowest in order to gain an understanding of the global spread of major 
languages.

### Scope

World Data Analysis System

### Level

User Goal

### Preconditions

Language data is available in the database.

### Success End Condition

A report containing population of Chinese, English, Hindi, Spanish, and Arabic speakers worldwide is available to the
data analyst.

### Failed End Condition

No report is produced.

### Primary Actor

Data Analyst.

### Trigger

Data analyst requests report of on population of major language speakers worldwide.

## MAIN SUCCESS SCENARIO

1. Data analyst requests language speakers report.
2. Report is generated accurately.

## EXTENSIONS

None.

## SUB-VARIATIONS

None.
